Rede de computadores

uma rede de computadores é um conjunto de equipamentos interligados de maneira a trocarem informações e compartilharem recursos, como arquivos de dados gravados, impressoras modems softwares e outros equipamentos.

importância das redes de computadores

 Proporcionando economia, compartilhamento de recursos, como impressoras e projetores, bem como de serviços, que é o caso da internet. permite também, por meio do compartilhamento de arquivos, o trabalho em equipe dos colaboradores.

LAN WAN

LAN(Local Area Network) é uma rede de computadores que cobre uma área limitada, geralmente um edificio ou um campus. É usada para conectar dispositivos em um espaço fisico limitado, como uma casa, escritorioou sala de aula. já WAN (Wide Area Network) é uma rede de computadores que abrange uma area geografica maior, geralmente uma cidade, um estado ou até mesmo um país. Ela é usada para conectardispositivos que estão fosicamente distante entre si.
